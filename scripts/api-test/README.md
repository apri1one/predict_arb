# API test scripts

These are lightweight scripts to benchmark the Predict and Polymarket APIs before building trading modules.

## Setup

1) Copy `../../.env.example` to `../../.env` and fill values.
2) Run scripts from repo root.

## Scripts

- `scripts/api-test/predict_bench.ps1`
- `scripts/api-test/polymarket_clob_bench.ps1`
- `scripts/api-test/polymarket_condition_to_tokens.ps1`
- `scripts/api-test/polymarket_ws_smoke.ps1`
